Dynamic Web Project에서 외부 라이브러리 사용하는 방법 (택 1)

1. Tomcat에 라이브러리 등록하기
	1) CATALINA_HOME 열기
	(그 전에 제일 먼저 jar 폴더에서 사용할 jar를 복사!)
	2) lib 디렉터리 열기
	3) 사용할 외부 라이브러리 복붙
	4) 해당 Tomcat을 사용하는 모든 컨텍스트에서 등록된 라이브라리 사용
	
2. 특정 컨텍스트(프로젝트)에 라이브러리 등록하기(그 프로젝트만 동작시키는 방법. 수업에서 진행할 방식은 2번!)
	1) src/main/webapp/WEB-INF/lib  (=> web root를 열고 WEB-INF를 열고 lib 열기)
	2) 사용할 외부 라이브러리 드래그(복사)
	
	
	

==> 일반적으로 라이브러리는 특정 컨텍스트용(2의 방법)으로 받아 쓴다
