cat ex.txt  ---> cat : 파일에 저장된 정보 확인 가능
                               (# 은 주석. 실행되는 부분 x )

rm ex.txt(파일명) ---> rm : 파일 제거
rm -i : 파일 제거할거냐고 묻는다 (y or n)
alias rm='rm -i'    rm이라고 적으면 rm -i가 동작하도록 별칭준다.
alias :  등록되어 있는 alias 다 보여줌. 하지만 내가 따로 저장한건 푸티(리눅스)를 껐다 키면 다시 리셋됨
         -▶ 리셋되지 않게 .bashrc 파일에 등록해놓기
 [ vi .bashrc  ->  Go(마지막 줄에 입력하기) -> alias rm='rm -i' ->  esc키 +  :wq ]    -> 리눅스 껐다켜도 alias 검색해보면 살아 있음!
 

* 편집기로 파일 열기 = vi : 파일명
vi : 파일명  ->  명령모드 (로 진입. 명령모드에서는 편집 안됨)
수정/삭제 등 편집하고 싶으면 '입력모드'로 빠져야 됨.
명령모드 --- a, i, o ---> 입력모드(로 들어가서 작업을 하고!)
a = 커서 뒤에
i  = 커서 앞에
o = 커서 아래
입력모드 --- esc키 ---> 명령모드(로 다시 돌아오기)

* 명령모드 --- : ---> 콜론모드 ( 콜론모드(last line mode) : 마지막 줄로 이동 )
 > esc키 누른 후에  :  <
:wq  = 저장하고 닫기
:w 파일명  = 다른이름으로저장
:q   = 그냥 나가기
:q!  = 입력모드에서 작업한거 저장 안하고 나오고 싶다
콜론모드 --- 아무거나 눌러 ---> 명령모드
(참고, 콜론모드에서 입력모드로 넘어가는 방법은 없다)

명령모드->입력모드
 - 방향키 or 좌우 ( h , l ) , 상하 ( k , j ) : 커서 이동 가능
 - $ : 맨 뒤
 - ^ : 맨 앞
 - G : 맨 마지막 줄
 - Go : 젤 마지막 줄 밑에 o 새로 추가하고 싶을 때.
     └>  이제 입력모드 온 것! 
 - u : 접속해서 한 모든 것을 취소(undo 같은 기능)
